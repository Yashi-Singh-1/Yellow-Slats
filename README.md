<h1>Yellow Slats</h1>

<h2>Challenge Description</h2>

<p>In this challenge, you will create a stylish and centered layout using HTML and CSS, focusing on precise positioning and layering of elements. The design consists of a main container divided into two lines, each containing two blocks with specific placements and decorative elements.</p>
